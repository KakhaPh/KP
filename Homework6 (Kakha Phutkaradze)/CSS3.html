CSS 3

CSS 2 ვერსიისაგან განსხვავებით, რომელიც სხვადასხვა შესაძლებლობების აღმწერი ერთი დიდი სპეციფიკაციაა, CSS 3 დაყოფილია რამდენიმე ცალკეულ დოკუმენტად — „მოდულად. თითოეული მოდული ამატებს ახალ შესაძლებლობებს, ან აფართოვებს არსებულს, რომელიც აღწერილია CSS 2-ში, უკუთავსებადობის შენარჩუნებით. მუშაობა CSS 3 დონეზე დაიწყო CSS 2 დონის რეკომენდაციის გამოქვეყნების დროს. CSS 3-ის ყველაზე ადრეული პროექტი გამოქვეყნდა 1999 წლის ივნისში. მოდულურობის გამო, სხვადასხვა მოდულს გააჩნია განსხვავებული სტაბილურობა და სტატუსი. ზოგიერთ მოდულს აქვს რეკომენდაციის კანდიდატის (რკ) სტატუსი და ითვლება შედარებით სტაბილურად. „რკ“ სტადიაზე ვენდორებზე გაიცემა მწარმოებლის თავსართის მოცილების რეკომენდაცია.
